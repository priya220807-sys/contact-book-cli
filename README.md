# Command-Line Contact Book Application

A lightweight Python command-line interface (CLI) application that allows users to manage a list of contacts. The data is persistently saved using an external text file layout (`contacts.txt`). This project also includes a complete collection of solved Python core foundational exercises.

##  Features
- **Create (Add Contact):** Saves a new contact name and phone number dynamically to the text database.
- **Read (View & Search):** Displays all saved contacts systematically or performs case-insensitive matching to find specific contact details.
- **Delete Contact:** Safely removes selected profiles from the registry and overwrites the text file to save changes.
- **Core Python Work:** Includes 26 fundamental Python logic exercises covering conditionals, loops, functions, prime number generation, and math algorithms.
